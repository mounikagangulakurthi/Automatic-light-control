# Automatic-light-control
# Motion Sensor Based Smart Lighting System

## Overview
This project uses a PIR Motion Sensor to automatically control lighting based on human movement.

## Features
- Motion Detection
- Automatic Light ON/OFF
- Energy Conservation
- Smart Home Automation

## Components Used
- PIR Motion Sensor
- Arduino Uno
- LED
- Resistor
- Battery

## Working Principle
The PIR sensor detects motion and sends a signal to the controller. The LED turns ON when motion is detected and turns OFF when no motion is present.

## Applications
- Smart Homes
- Offices
- Corridors
- Energy Saving Systems
